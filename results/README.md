# Outputs

All outputs generated by the tutorial are written here. Key artifacts:

- Alignment/BAM:
  - `ds.{sam,bam,sort.bam,dedup.bam,q20.bam}`
  - `xr.{sam,bam,sort.bam,dedup.bam,q20.bam}`
- BED and derived:
  - `ds.bed`, `ds.sorted.bed`, `ds.sorted.chr.bed`, `ds.sorted.chr.{plus,minus}.bed`
  - `xr.bed`, `xr.sorted.bed`, `xr.sorted.chr.bed`, `xr.sorted.chr.{plus,minus}.bed`
  - DS lesion centering: `ds.plus.10.bed`, `ds.minus.10.bed`, `ds.10.bed`
  - DS motif-filtered: `ds.dipy.{plus,minus}.bed`, `ds.dipy.bed`
- Counts/lengths:
  - `ds.len.txt`, `xr.len.txt`, `ds.dipy.count.txt`, `xr.count.txt`
- Coverage:
  - `ds.dipy.{plus,minus}.bdg/.bw`
  - `xr.{plus,minus}.bdg/.bw`
- Simulation:
  - `ds.sim.{bed,fa}`, `xr.sim.{bed,fa}`
  - Split: `xr.sim.{plus,minus}.bed/.bdg/.bw`
